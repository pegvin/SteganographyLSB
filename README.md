# MsgHide
Hide Text In Images Using LSB Stegnography Technique

```
Usage: ./msghide encode/decode [arguments]

[arguments]
   <filename>  Input File To encode/decode message into/from
   <message>   If Encoding, A Message To Encode Is REQUIRED
```
